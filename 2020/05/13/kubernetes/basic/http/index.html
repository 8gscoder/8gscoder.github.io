一、core struct

Handler

handler是一个能够处理request，然后写response的抽象处理器

ServeMux

ServeMux是一个保存着大量handler的处理器multiplexer（多路复用器，联想一下linux的epoll），能都接受request，然后按照注册好的handler进行做多路转发

注意 ServeMux也是一个特殊的Handler，只不过他是一个多路复用的Handler
