SSH & Proxy & 隧道

经常用ssh，但是从来没有好好的研究过ssh的协议原理，周末太无聊了，就看下这块叭。

ssh是一种能够在非安全的网络链路上，实现信息的加密、完整可靠传输。

ssh以非对称加密实现身份的验证，通常可以用密码登录和密钥登录。

ssh在tcp之下，这也是为什么ssh可以做转发隧道的原因，可以包装tcp/udp协议。

SSH的-L/-R/-D参数

-L [bind_address:]port:host:hostport

Specifies that the given port on the local (client) host is to be forwarded to the given host and port on the remote side.

正向代理，本地绑定一个端口，再指定一个host:hostport，所有访问本地这个port的流量都会通过这个ssh链接转发到对端的host:hostport上。

-R [bind_address:]port:host:hostport

Specifies that the given port on the remote (server) host is to be forwarded to the given host and port on the local side.This works by allocating a socket to listen to port on the remote side, and whenever a connection is made to this port, the connection is forwarded over the secure channel, and a connection is made to host port hostport from the local machine.

反向代理，指定对端port，host和hostport是本地可以访问的一个地址，那么链接建立完毕之后，对端主机访问这个port的流量就相当于转发到本地能够访问的host:hostport上。这是一种内网穿透技术，比如内网主机A和B和公网主机C，只有A可以访问公网，B不可以，如果B上有敏感数据的话，在A上进行-R把B的某些内网服务端口暴露给了C上的某个端口，那么外网就可以自由自在的进行访问B的内网服务了

-D [bind_address:]port

Specifies a local ``dynamic’’ application-level port forwarding. This works by allocating a socket to listen to port on the local side, optionally bound to the specified bind_address. Whenever a connection is made to this port, the connection is forwarded over the secure channel, and the application protocol is then used to determine where to connect to from the remote machine. Currently the SOCKS4 and SOCKS5 protocols are supported, and ssh will act as a SOCKS server. Only root can forward privileged ports. Dynamic port forwardings can also be specified in the configuration file.

socks5代理，相当于本地起一个socks5代理。所有流量配置了socks5代理转发到本地的这个端口，都会被当做是在remote主机上进行发包一样。

socks5和http代理的区别？

socks是一种协议是SOCKetS的缩写，位于会话层和传输层之间，socks5是发展过来是一个版本，socks5的代理是基于tcp的，简历

Serving 原理

Serving 总共有 5 个组件，其中 4 个在 knative-serving 这个 namespace 下面，是 controller 、webhook 、autoscaler、activator 这四个组件；还有一个 queue, 运行在每个应用的 pod 里，作为 pod 的 sidecar 存在。

\1. Controller 负载 Service 整个生命周期的管理，涉及、Configuration、Route、Revision 等的 CURD。
\2. Webhook 主要负责创建和更新的参数校验。
\3. Autoscaler 根据应用的请求并发量对应用扩缩容。
\4. Activator 在应用缩容到 0 后，拦截用户的请求，通知 autoscaler 启动相应应用实例，等待启动后将请求转发。
\5. Queue 负载拦截转发给 Pod 的请求，用于统计 Pod 的请求并发量等，autoscaler 会访问 queue 获取相应数据对应用扩缩容。

HTTP

http的长连接和短链接？

http是基于tcp的，所以http的长连接和短链接都依赖tcp的长短链接。

tcp是面向连接的，会进行三次握手四次回收建立socket连接，TCP本身没有长短连接的说法，完全是取决于上层协议怎么使用TCP

  • 短连接：每次传输都重新握手和挥手
  • 长连接：建立连接之后保持连接，不进行挥手操作，TCP协议里没有保活部分，但是OS都会进行keep-alive

而http的短链接在/2020/09/10/network/http/1.0之前都是默认短链接，在/2020/09/10/network/http/1.1之后默认都是Keep-alive的。Connection:keep-alive

文件系统

什么是文件系统

在类Unix操作系统之下，除进程之外，一切皆文件。文件系统是os对硬件设备硬盘使用的一种方式，linux下常见的是ext4，windows下是FAT/NTFS

文件系统的分层

内核为了屏蔽下层具体的FS实现，中间用了VFS来隔离，具体的FS只用实现VFS的接口，就能适配内核，将内核和FS解耦。

Sector、Block、Inode、file、file_operations

磁盘的存储最小单位是sector扇区，但是sector的单位512B，所以如果要找一个大的10Mb文件的话，需要读磁盘20480次，所以就有了「逻辑区块Block」，这样只要设置好Block和Sector的对应关系（1block=4sectors，4KB），就可以大幅度减少磁盘IO次数，但是越大就会越浪费，越小效率就越低。

inode是EXT文件系统用于记录block的属性（所属者、可读写执行、类型-/d/s/l/..、创建修改时间、字节数、file链接数、具体设备编号及设备指针）用的结构体，另外还会记录文件数据位于的block位置。

file才是linux记录进程实际打开文件的结构体，也就是常说的文件描述符表项，进程有个文件描述符表是[]file类型，而我们说的fd则是这个表的Index，系统调用open的实现很容易理解，入参是文件路径，调用内实现是在这个进程的文件描述表中新加一个file结构体，返回索引，inode会增加计数。

file_operations记录了设备读取的函数指针

设备文件devfs和udev

什么是设备文件？类unix操作系统都是基于文件概念的，所以对于os来说，增删设备也是抽象成文件，有统一的io接口来操作设备，而设备文件就是代表着一个“设备”，设备文件分为两种：块和字符，其实还有一种比较特殊的是网络设备。

  • 块设备：一般带有缓冲区，可以随机读取的，比如write的时候可以设置指针指向的位置，常见的就是硬盘（云硬盘）
  • 字符设备：一般都是有连续的数据流，带有顺序，不可随机读取的，常见的有键盘、串口、调制解调器
  • 网络设备：特殊设备的驱动，负责接收和发送数据帧，可能是物理帧也可能是ip数据包，取决于网络驱动，网络设备是net_device结构体，通过register_netdev注册到内核。和其他设备不通。

tty设备？pts终端？rzsz的原理？

  1. tty设备是一种特殊的字符设备，对应着物理显示器设备，大家都知道，最初的计算机由于价格昂贵，因此，一台计算机一般是由多个人同时使用的。在这种情况下一台计算机需要连接上许多套键盘和显示器来供多个人使用。但是其实现在已经很少有真实的物理tty设备了，基本都软件仿真终端，都是在内核实现的。

  2. pts终端就是伪终端

    为了便于将终端仿真移入用户空间，同时仍保持 TTY 子系统(TTY 子系统指 TTY 驱动和行规范)的完整，伪终端被发明了出来(pseudo terminal 或 pty)。伪终端在内核中分为两部分，分别是 master side 和 在 TTY 驱动中实现的 slave side。注意上图中的 xterm，这是一个运行在用户态的终端仿真程序，比如 Ubuntu Desktop 中的 GNOME Termina

  3. 那我们通过ssh进行的rzsz又是个啥鸡巴原理呢？

    ​ 网上找了一张图，很清晰~

    当使用本地的Terminal连接到ssh服务之后，ssh-server会fork一个shell，这个shell会关联一个pts虚拟的终端设备，ssh-server会接受到终端的键盘输入，然后写到对应的pts设备，shell收到对应的stdin之后会有输出，输出的结果会被ssh-server通过网络协议传输回对应的terminal的ssh客户端

云硬盘挂载？什么是挂载？

云硬盘是一种虚拟块设备，和虚拟机配合使用，挂载到虚拟机的指定目录下。

Netlink Socket

Netlink socket是一种kernal和user space之间通信的一种方式，也是一种IPC，注意，netlink socket不是网络socket，只可用于主机内的进程通信，socket是通过进程pid来标志的。

其他常见的kernel和user space之间通信的方式有sysctl、系统调用、procfs和ioctl，但是都是一种user->kernal的方式，且是同步、单工通信。

而netlink socket协议族则更加灵活，且可以自己定制protol参数，通过insmod命令插入内核即可。

​ “哦我威海的，嗨呀，我是个船员儿，之前在镇江带了几天，再后来从威海要去深圳港口归队”

​ “出海是不是都出国境了？”，我一听说船员吓一跳，出海大概是出国了吧，现在国外这么严重。

​ “是的，出了，但是就没上过岸，床上就我们几个船员，和这大巴车上一样，五六个人。”，威海大叔说道。“这广东zf没事找事做，搞什么粤康码就是在骗钱，这下事情可遭了，在深圳这边住酒店14天得个大几千呢啊，这个是笔不小的钱啊”

​ 我心里：想嗨呀zf就算再坑你，也不会想坑你这点钱啊，而且十四天咋可能得大几千，也就两三千吧。

​ 后来我心态也没了，整个人都蔫了。

​ “这空调温度也太低了吧，想冻死人吗”，一个同车的连云港的说道。

​ 后来我听见坐在车头的女安保在问空调能关吗？司机说关不了，要不开窗户吧，车外有热风。“再坚持一下，马上到了，很快的”，女安保说。

​ 车内窗户打开，外面的热浪和大巴车内的冷气混合着，但是我还是觉得冷气更胜一筹。

​ 等了好久才到隔离酒店。到了之后，发现一个看起来有点老实的那种酒店环境，有个院子，院子里的人有穿着防疫服的，有穿着警服的，有便衣的。被安排下车的时候，门口的保安似乎是在躲着我们，往后退了几步，我不确定是否是我太敏感了，我很希望是。

​ 在院子的左边支起了棚子，坐着几个医护人员吧，穿的很厚的防疫服，跟电视机里新闻的一样。我坐下来，黑龙江的女的又开始疯狂bb，哭腔又来了，还是那一套说辞。录信息的医护人员好像都已经听习惯了，就一直点头听着，然后问一些信息。

​ “你叫什么名字多少岁了”，我对面的女医护说。

​ 我回答了之后又是一大篇问题。

​ “心理状况如何？良好吗？” 她说。

​ 我犹豫了三四秒。“良好”。

​ “之前有过自残行为吗？”

​ “没有”。

​ “对本次防疫措施有什么看法吗？”，

​ 我看了下选项有：1. 完全理解；2. 部分理解，有不认可的地方；3. 完全不理解。医护的笔已经放在1上了准备勾了。

​ 很激动，想张口说3，但是略怂，语气大了点说这个我选2。

​ “曾经有没有对人生感到有很大的挫败感？”

​ “我觉得我现在就很失败的”。

​ 小医护笑的不行，说哎呀这有啥这就失败了，我们这医院还没遇到过阳性的呢。

​ 我心里笑了下，但是假装没感情，吓一吓她。

​ 然后要手机进行注册一些东西，我手机没电了，就去问了下机场的女安保。“姐，我手机没电了，1%，一会儿我怕人家不给我说话，能帮我问下有没有ios的充电线吗？”。然后她跑去大厅问了下人，然后跑回来说，“没事的房间都有充电头。你现在手续搬完了去大厅办入住吧。”

​ 我一脸sm脸，然后到大厅有个小哥过来说“请把你的包放在这个上面过安检”，我心里想，滚吧，老子就不知道这是要过安检的。

​ 后来我放上去了，才发现吗的，他过完安检要开包。他很温柔的给我说“您好，我们这边是要搜一下您的包的，能麻烦打开给我们看下吗”

​ 我知道这很过分，So what? What can I do? 我打开了书包拉链。

​ “我们这里是不允许带一些特定药物进去的，还有一些管制刀具啊一些尖锐的，我看下有没有哈”，然后他拿出我的发蜡问了句这是什么哈，我说这不是药，这是发蜡。然后我突然想起来我带了周五要吃的甲氨蝶呤，就问了下他，他看了下之后叫我等一下，需要问下医护，然后跑出去问了下刚刚的医护，没多久，医护跑了进来问了句，你是怎么了啊为什么吃这个药？

氨甲蝶呤（英语：Methotrexate，又称甲氨喋呤、甲氨蝶呤、氨甲喋呤；在台湾，学名为胺基甲基叶酸，商品名为灭杀除癌锭及治善锭。）氨甲蝶呤是一种化疗药物和免疫抑制剂[5]，被用来治疗癌症、自体免疫疾病、子宫外孕和进行药物流产[5] 。可治疗的癌症类型包含乳癌、白血病、肺癌、淋巴癌和骨肉瘤[5] 。可治疗的自体免疫疾病包含银屑病、类风湿性关节炎和克隆氏症[5] 。使用方式为口服或注射[5] 。）

​ 我从她眼神里面看出来有一点害怕和同情。“我有关节炎”，我就知道她肯定会很慌张。

​ “哦哦，这个药给他吧”。

​ 我拿着要到柜台办理入住，虽然隔着笨重的防护服，但是我明显能看出来柜台的医护年级应该比我小啊，而且我能明显看到她的手在抖，是真的在抖。我突然心里意识到，这次疫情真的很严重，已经严重影响人们的正常生活了，我突然也有一点点开始理解为什么不管什么原因红码都要隔离了（虽然我的码是真冤枉）

​ “em..你好你房卡办好了，费用是200一天，需要您支付14天的房钱，另外酒店提供的早午晚饭一天是100块，这个您自己支付，也可以自己点外卖，我们会帮您送上去，外卖不允许有易拉罐、烧烤签这样的尖锐锋利物品，您看能接受吗？”

​ “嗯”。

​ 之后就来到房间入住了。

​ 房间真的是老式的酒店，无论是大屁股的电视机还是床头那个时间错了显示着“2004.1.1”的电话，都让我感觉很不舒服。3点的夜，外面院子里灯还亮着，我站窗边看了好久，看着威海大叔还在跟着院子里的可能是领导的公安争执着，但是还是被送进大厅办入住。

​ 我想洗个澡，进入卫生间，发现酒店内什么都没有，尤其是灯光也是疮黄的，让我心里很不舒服。

​ 我决定不洗澡了，头也很重。躺在床上，心里没有怨恨，没有担心，没有迷茫，当然也没什么希望（我心理状况还ok，只是可能很累，心里什么都没想）。

​ …

​ 第二天早上，我头不知道为什么变得更疼了（就是比有点疼还疼，但是不是很疼），我拿起手机看了下，发现周一了，企业微信又是一大堆消息。心里很难受也很烦躁，小怡也在一直说啥（小舒怡你大爷的哦）。看到骚骚的消息说要联系溧阳市第九军区空指部炸掉宝安机场，我笑了。

​ 门铃突然响了。

​ “嘴张开，啊..”，昨天那个在机场说没人情的医护来做核酸检测了，我认得她，包着防护服我也认识她那双刻薄的眼睛。我从她眼睛里也看出来她认得我了。测完后，我瞪了她一眼，就关门了。

​ …

​ 在房间处理了一下工作的事情。网太差了，动不动就断线，很烦躁。不断的问酒店的医护，我核酸结果什么时候可以出来啊。

​ …

​ 下午的时候微信上收到酒店的消息，说我是阴性，收拾行李可以下来了。

​ // TODO wechat pic

​ 我收拾东西办了手续，拿着核酸检测报告，印了手指印，走出酒店门口。

​ 我问了个公安的，“你们包接不包送吗？”

​ …

​ 下午五点多的华茂航空酒店大门口的树荫凉…嗯，跟tm没有一样热。

​ 我准备打车回了。滴滴了一下南山区xx花园。预计70块。车来了之后上车了，刚准备关门，看见威海的大叔。他看见我说兄弟我这没打过滴滴，这附近没出租，能拼一下车不。

​ 我心里想算了，即使让正常的人来决策，肯定都是不让他上车的，特殊情况。但是我还是说，“嗯，可以”。因为你们没有我的经历。

​ 在之前，我一直都觉得哇靠靠，中国牛逼，这次疫情在中国这么快就被抑制住了，就应该强制隔离，就应该这样！但是当我是个被强制的时候，心里也有点不是滋味。

​ …

​ 车上，威海大叔和我聊天，还在吐槽这zf管他的事情。说这就是在骗他的钱，

​ 我问“你交了2800了没”

​ 他说“我没交，我就交了200，他能退给我就退，不能退给我就算，就当住酒店了。这江苏的苏康码真他妈操蛋，我从昨天上飞机之前到现在就吃了一碗7块钱的泡面，真是害死人。你昨天晚上睡着了没？”

​ “我睡着了，我前天喝了点酒头很疼”。

​ “我这一宿没睡，我一晚上给镇江政务局、114、12345、还有威海的都打了电话，没几个接的，我这一晚上都没睡着觉，就怕这今天出不来耽误我归队啊”

​ “啊好吧”

​ “这打车钱怎么算啊，我给你打一半的钱吧，唉师傅，能给我打车的发票不”。

​ 滴滴司机：“现在在去南山区xx花园，你和这兄弟协商下看怎么搞吧，滴滴只能打电子发票。”

​ 我说：“我加你微信吧，我到了xx花园修改目的地，给你送到蛇口的港口酒店。之后我把发票发给你，你打一半钱给我”。

​ “好好”。他说。

​ “其实你都可以坐着我打的车到蛇口，然后删了我微信。”。。其实他难道没发现他可以打着我的车然后下车删了我的微信不给我钱吗？

​ “嗨，咋能这样，我们在社会上做事的，都是靠的诚信，怎么还能骗你呢？”

​ 出租车司机说：“这样，你到蛇口起码得个一百多，你给这小兄弟打一百块钱，到了之后多退少补？”。（内心OS，那这样岂大叔请我坐车了。。）

​ “也可以吧，兄弟你一定要把发票给我发过来啊，我这得找公家报销呢”，“我们北方人就是不绕弯，讲诚信，当时我去这个单位就是因为他不要我xx证明”（忘记啥证明了）

​ …

​ 我到了xx花园之后下车回到了家。

​ “滴滴滴，密码正确请进”。

​

​ 这三四天过的我真的心力憔悴。回到家还是打开电脑记录了一下。

​ 这几天的经历之后，成长了不少。就是这一路上遇到形形色色不同的人，包括未来可能遇到更多不同于我日常生活密切相处的人，构建出来我对整个社会整个人生的价值观。
